 8 Week SQL Challenge – SQL Case Studies

 Project Overview
This repository contains my solutions for all 8 Case Studies from the 8 Week SQL Challenge.  
The goal is to practice SQL querying, data analysis, and problem-solving using real-world inspired datasets.

 Tools & Technologies
- Database: Oracle SQL  
- SQL Client: SQL Developer  
- Dataset Source: [8 Week SQL Challenge](https://8weeksqlchallenge.com)

 Project Structure
/data → SQL scripts to create tables & insert data
/queries → SQL solution queries for each case study
/results → Screenshots of query outputs (optional)
README.md → Project documentation

 Case Studies Included
1. Case Study 1 – Danny's Diner  
   Analyze restaurant orders, menu items, and customer behavior.  

2. Case Study 2 – Pizza Runner  
   Explore pizza orders, delivery patterns, and top customers.  

3. Case Study 3 – Foodie-Fi  
   Analyze subscription behavior, churn, upgrades, and plan trends.  

4. Case Study 4 – Data Bank  
   Work with financial datasets to analyze transactions and customer patterns.  

5. Case Study 5 – Data Mart  
   Focus on data aggregation, sales trends, and inventory analysis.  

6. Case Study 6 – Clique Bait  
   Analyze social network interactions and user engagement.  

7. Case Study 7 – Balanced Tree Clothing Co.  
   Explore sales, products, and customer trends in a clothing company.  

8. Case Study 8 – Fresh Segments  
   Segment customers and analyze purchasing behavior for marketing insights.

 How to Run
1. Open SQL Developer  
2. Create a new database connection  
3. Run the scripts in `/data` to create tables and insert data  
4. Run queries from the `/queries` folder  
5. Review results in `/results` (screenshots optional)

 Learning Objectives
- Practice SQL SELECT queries and JOINs  
- Use aggregations, GROUP BY, and window functions  
- Analyze real-world datasets  
- Improve problem-solving and data interpretation skills  
- Generate clear outputs and visualizations
