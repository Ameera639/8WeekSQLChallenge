# 8 Week SQL Challenge - Danny's Diner Case Study

## Project Description
This project contains my solutions for the 8 Week SQL Challenge – Case Study #1 (Danny's Diner).
The goal is to practice SQL querying, data analysis, and problem-solving using a sample restaurant dataset.

## Tools & Technologies Used
- Database: Oracle SQL
- SQL Client: SQL Developer
- Dataset Source: 8 Week SQL Challenge

## Project Structure
/data       → SQL script for creating tables & inserting data  
/queries    → My SQL solution queries  
/results    → Query outputs screenshots (optional)  
README.md   → Project documentation  

## How to Run the Project
1. Open SQL Developer
2. Create a new connection
3. Run the script in /data to create tables
4. Run queries from /queries folder
5. Review results
